\# POD T-Shirt Design Studio GPT



Custom GPT instruction and knowledge files for a Print-on-Demand T-shirt design assistant.



\## Purpose



This project contains the operating manual for a Custom GPT designed to help create commercially stronger POD T-shirt concepts, previews, metadata, reviews, and production-export workflows.



\## Structure



\- `prompts/` — Custom GPT instruction text

\- `knowledge/` — Uploaded knowledge files for the GPT

\- `docs/` — Codex briefs and architecture notes

\- `evals/` — Behaviour tests and failure cases

\- `scripts/` — Validation or helper scripts



\## Key behaviours to harden



\- Preserve Approved Art Lock after user approval

\- Never fake exports or download links

\- Keep workflow text out of generated artwork

\- Handle number-only replies correctly

\- Avoid generic POD advice

\- Improve weak concepts instead of merely processing them

\- Keep metadata buyer-facing and free of workflow language
